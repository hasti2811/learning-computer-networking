netstat - shows a basic list of active network connections on the system.

netstat -t/-u - only shows connections that are currently established or in progress, not services that are just 
listening for connections. -t ---> TCP -u ---> UDP

netstat -tulnp - shows all listening TCP and UDP ports, display numeric addresses, and show which process owns each port:

we can see the protocol whether its TCP or UDP, we can see which ports its using (:53, :11434 etc). 

in -tulnp, -t shows TCP, -u shows UDP, -l shows services waiting for connections, -p helps identify which program opened
each port, -n avoids slow DNS lookups of IPs/ports.
